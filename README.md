# Sales & Profit Dashboard

## Project Overview
This interactive Power BI dashboard analyzes sales and profit performance to help businesses monitor KPIs, identify trends, and make data-driven decisions.

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Features
- Sales KPIs
- Profit Analysis
- Regional Performance
- Product Performance
- Interactive Filters & Slicers
- Time-based Trend Analysis

## Business Value
- Track sales performance.
- Identify high-performing products.
- Compare profitability across regions.
- Support strategic business decisions.
